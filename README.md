 ## Get Started
 
 #### 1. Clone the Repo
 
 On the command prompt run the following commands
 ```sh
 $ git clone https://github.com/Doha26/Facebook-React-native.git
 
 $ cd facebookUI
 
 $ npm install
 
 $ react-native run-ios
 
 ```
